## Task 1 – Brief Description

For Task 1, I created a Digital Literacy Awareness infographic using Canva. The infographic explains the meaning of digital literacy and its importance for college students. It covers key areas such as safe internet practices, useful digital tools, and maintaining a professional online presence. I included points about protecting personal information, using strong passwords, verifying information before sharing, and using platforms like GitHub and LinkedIn for academic and professional growth.

The infographic was designed to be simple, visually clear, and easy to understand. I used icons, headings, and short bullet points to organize the content. This task helped me learn how visual communication can effectively spread awareness about digital literacy among students.
